# ZenMIDI
EN ///
Easy Powerful Program, That Can Modify MIDI!
Possibilities:
Easy UI That Can Understand Any People!
Working without internet, not need hard settings
Removing All CC Events!
You can download the ready-made .exe file without assemblies!
How To Use:
Download .exe from releases
Open EXE
Select a Output PATH
Select a MIDI
And Then You Can Modify!
